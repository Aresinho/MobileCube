MobileCube
Version 3.1.0 (BETA) - Build #062010A

Welcome to MobileCube 3.1.0(BETA)! We hope you enjoy this theme very much so let's install it shall we?? All you have to do in order to install it is follow these simple instructions and you will have your very own MobileCube theme up and running in no time!

Instructions

  • 1. Unpack MobileCubeBeta#062010A.zip

  • 2. There are two folders in the folder you unpack. "MobileCube" and "calendar".Move the "MobileCube" folder into the "skins" folder of the Roundcube Directory. Such as "/roundcube/skins/".

  • 3. Move the "calendar" folder to the plugins directory. Such as "/roundcube/plugins/".

  • 4. In the RoundCube Config file (/roundcube/config/main.inc.php) set your skin as "MobileCube"

    • To do this find "$rcmail_config['skin'] = 'Default';" (minus quotes) and change where it says Default to MobileCube. If there is something else there besides Default change that to MobileCube.

  • 5. In the RoundCube Config file (/roundcube/config/main.inc.php) add "calendar" to the list of active plugins. It should look something like this. "$rcmail_config['plugins'] = array('calendar');" (minus quotes)

  • 6. Now in the "calendar" folder there is another folder called "SQL". Take the file in the "SQL" folder and run it in your roundcube database. You can use a program such as PHPMyAdmin to do this. Open your roundcube database and run this file in it.

  • 7. Remove the "SQL" folder that is located in the "cubecalenar" folder. (Such as; remove /roundcube/plugins/calendar/SQL/)

  • 8. You may now login and enjoy your theme! Forupdates of the theme go under settings and on the right bar it will inform you of the most current version.
